# Documentation Assets

This folder contains images and videos used in the main README.md.

## Adding Screenshots

Place your screenshots in the `images/` folder with these names:

- `homepage.png` - Homepage with language selection
- `ingredients.png` - Ingredient selection screen
- `recipes.png` - Recipe swipe cards view
- `recipe-detail.png` - Full recipe details (ingredients + steps)
- `favorites.png` - Favorites grid view

## Adding Videos

Place demo videos in the `videos/` folder:

- `demo.mp4` - Full application walkthrough

## Image Guidelines

- **Format**: PNG for screenshots, MP4 for videos
- **Size**: Maximum 1920x1080 for screenshots
- **Compression**: Use tools like TinyPNG to reduce file size
- **Naming**: Use lowercase with hyphens (kebab-case)

## Taking Screenshots

### Recommended Tools

- **macOS**: Cmd+Shift+4 (native), or use CleanShot X
- **Windows**: Snipping Tool, or Greenshot
- **Linux**: GNOME Screenshot, Flameshot

### Best Practices

1. Use consistent window size (1440x900 recommended)
2. Show realistic data (not Lorem Ipsum)
3. Capture key interactions and features
4. Include both desktop and mobile views if applicable
5. Highlight important UI elements

## Recording Demo Videos
